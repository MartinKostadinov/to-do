About To-do app

This is responsive static to-do application. It's created with HTML5, CSS3, Bootstrap, jQuery and Handlebarsjs. The icons used are - material icons from google;

There are 3 options for user customization - background color, title color and screen position(only for descktop devices). More to come soon !

User can create as many as he wants categories. For each category there is task counter.
